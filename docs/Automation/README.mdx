## Overview
<div style={{ borderBottom: '1px solid #000', marginTop: '20px', marginBottom: '20px' }}></div>

The purpose of this guide is to provide an overivew of the different automation scripts to download the differnt components required for a an end-to-end wca4z install.

**We will cover:**

- System Pre-Flight checks: The hardware and software requirements needed to carry out the install
- Refactor Installation
    - pre-flight system check to ensure all the required OS system and hardware requirements are met
    - Tool installation
       - Podman
       - Refactor Containers
- ADDI Installation: A walkthrough of the different prerequisite installs along with the ADDI install itself
    - Set necessary User permissions
    - Java Runtime Environment install
    - SQL server install and permissions
    - ADDI Installation
        - TLS and Certifcate configurations
        - DB2 Cloud Database Config
